Draft Design Web — Landing Premium Black & Gold
Este repositorio contiene la landing premium de Draft Design Web, diseñada con un enfoque boutique: estética Black & Gold, microinteracciones elegantes y una experiencia visual cuidada al detalle.

Características principales
Diseño Black & Gold con una estética elegante y minimalista.

Menú premium con subrayado dorado ultra fino y animaciones suaves.

Efecto dorado lateral en el logo, discreto y sofisticado.

Animaciones Reveal on Scroll mediante IntersectionObserver.

Botón “Volver arriba” que aparece de forma inteligente al llegar al footer.

Diseño responsive con menú móvil accesible y optimizado.

Código modular, limpio y organizado.

Tecnologías utilizadas
HTML5

CSS3 (animaciones, efectos premium, responsive)

JavaScript Vanilla

Phosphor Icons

IntersectionObserver API
